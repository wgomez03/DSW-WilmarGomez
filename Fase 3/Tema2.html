CSS Grid

¿Qué es CSS Grid?


Se trata de un modelo de maquetación CSS con base a una rejilla, teniendo en cuenta
que ahora Grid es un estándar y no es necesario ajustar mucho el código para aplicarlo
desde el archivo CSS se emplea por medio de nuevos atributos, valores y sintáxis, gracias
a esto, es posible emplearse en diversos sitios estáticos, programación con lenguaje
determinado, framework o cualquier otro dispositivo. (García, 2018)

El CSS grid ofrece diversas características en los elementos de la página, tales como:
- Control total de las dimensiones
- La ubicación de los ítems es libre
- Permite colocar los ítems donde se requiera

¿Por qué utilizar CSS Grid?


Porque permite lograr muchos diseños diferentes de una página Web, dado que es posible
dividir en diversas áreas o regiones la página, definiendo las regiones en relación de
tamaños, posiciones y capas entre partes de un control construido a partir de primitivas HTML.
Gracias a su uso, el autor puede alinear los elementos en filas y columnas de una manera más
sencilla que con el uso de tablas, permitiendo también que los elementos se solapen y superpongan. (MDN, 2021)
